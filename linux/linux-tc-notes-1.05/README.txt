Linux-tc-notes
--------------

  Notes on the Linux Traffic Control Engine.  These are
  currently simple ASCII text files, viewable with just
  about anything.

  They can also be viewed at the home page:
    http://linux-tc-notes.sourceforge.net/


Installing
----------

  Packages are available for Debian and RedHat style
  distributions at the home page.  If you install using
  one of them you can skip this section.

  To install copy the files to a place of your choice.
