Linux-tc-notes
Notes on the Linux Traffic Control Engine

The Linux traffic control engine implements Quality of Service for network traffic flowing through the Linux box. These are notes I made when I first used it, made mostly because there no detailed documentation was available. They are dated, but sadly still useful because for things like U32 and HFSC the situation remains the same to this day.
